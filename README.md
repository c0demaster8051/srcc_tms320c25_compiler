# SRCC - small (simple), retargettable C compiler

Данная версия изначально делалась под сигнальный процессор TMS320C25, но потом, пока не стало
слишком поздно, решил сделать компилятор более универсальным. Он уже выдаёт рабочий код
и поддерживает типы int и char, сейчас находится в процессе добавления long и float.

Пока умеет только самое необходимое: условия if-else, циклы while, массивы и указатели.
Из низкоуровневого - может работать с портами, можно вручную задавать адрес размещения
отдельных глобальных переменных (для работы с периферией).

Сначала компилирует программу в байт-код, который можно исполнить прямо внутри компилятора,
далее функция из target.c транслирует его в ассемблер целевой архитектуры и выдаёт
текстовый листинг. Для получения бинарника используется telemark assembler,
который легко ищется на сайтах old-dos и подобных. Формат листинга так-же заточен под
этот конкретный ассемблер.

#### Сборка:

```sh
gcc -o srcc srcc.c
```

#### Посмотреть help:

```sh
srcc -h
```

#### Исполнить тестовый код на VM:

```sh
srcc test.c
```

#### Собрать его под tms320c25:

```sh
srcc -t test.c -cs 0x10 -ds 0x700 -ss 0x800
```

Полученный бинарник можно прогнать в отладчике, который так-же можно найти среди моих проектов.
